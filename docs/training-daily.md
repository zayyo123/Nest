# Nest 学习型教学项目 - 每日日历（第一轮，8 周计划的每日任务概览）

- 说明：以下为第一轮教学任务的日历化拆解，便于你按日推进学习。
- 每日任务安排可能因个人进度调整，建议每天 30–90 分钟学习与实践。
- 目标：确保每天有可执行的小任务，积累成完整可运行的全栈应用。

Week 1
- Day 1: 环境确认与基础 API 框架搭建，后端 /health、/docs 路由初步要点
- Day 2: 完成 NestJS 的 /auth/register、/auth/login，JWT 演示与 Swagger
- Day 3: 完成 Project/Task 的初始实体与 CRUD 架构设计
- Day 4: 后端 CRUD 的简单测试端点（Postman/Swagger）
- Day 5: 前端 Vue 3 + TS 项目骨架创建，路由与登录界面
- Day 6: 前端调用后端 /auth/login、/auth/register 的接口
- Day 7: 评审与本周回顾

Week 2
- Day 8-10: 完成 Projects 的前端全功能页面（列表、创建、编辑）
- Day 11-13: 完成 Tasks 的前端全功能页面（列表、创建、编辑、过滤、分页）
- Day 14: 集成测试（API Swagger 验证、前后端联调）

Week 3-8: 见 training-plan.md 的阶段性计划
